# Flyora ✈️

A full-stack Flight Booking System built using the MERN Stack.

## Features

### User Features

- User Registration & Login
- JWT Authentication
- Search Flights
- Flight Details Page
- Book Flights
- Multiple Passenger Booking
- Cancel Booking
- Delete Cancelled Booking
- Razorpay Payment Integration
- My Bookings Page
- Flyora AI Assistant (Gemini API) for flight-related help and quick user support

### Admin Features

- Admin Login
- Dashboard
- Create Flight
- Edit Flight
- Delete Flight
- View All Bookings
- Revenue Statistics

## Tech Stack

### Frontend

- React
- React Router
- Tailwind CSS
- React Hot Toast

### Backend

- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT
- Razorpay
- Gemini API (Flyora AI Assistant)
## Installation

### Frontend

npm install

npm start

### Backend

npm install

npm run dev
